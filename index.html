Brief Basketball History

Basketball was invented in 1891 by James Naismith as an indoor winter sport. It evolved from peach baskets and a soccer ball into a worldwide game with professional leagues like the NBA.

How to Play: Basics

This section includes an ordered list with an unordered list embedded inside it.

  1. Understand the Objective
    • Score more points than the other team.
    • Make baskets by putting the ball through the hoop.
    • 2 points inside the arc, 3 behind the arc, 1 per free throw.
  2. Learn the Core Rules
    • Dribble to move‚Äîdon‚Äôt travel.
    • No double dribble‚Äîdon‚Äôt stop and start dribbling again.
    • Play defense without unnecessary contact (avoid fouls).
    • Use spacing and passing to get better shots.
  3. Practice the 4 Core Skills
    • Dribbling (control + change of direction)
    • Shooting (form + repetition)
    • Passing (chest pass, bounce pass, overhead)
    • Defense (stance, footwork, positioning)
Beginner tip: Start with layups + dribbling with both hands.
